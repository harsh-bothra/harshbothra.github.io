Application Security Testing Standards & Frameworks

OWASP Projects

OWASP Top Ten (Web)

Official: owasp.org/www-project-top-ten/

Lists the ten most critical web application security risks, updated based on real-world data.

OWASP Mobile Top Ten

Official: owasp.org/www-project-mobile-top-10/

Highlights the top mobile application risks for Android and iOS platforms.

OWASP API Security Top Ten

Official: owasp.org/www-project-api-security/

Focuses on API-specific vulnerabilities, such as broken object-level authorization and excessive data exposure.

OWASP ASVS

Official: owasp.org/www-project-application-security-verification-standard/

Defines security requirements and controls across multiple verification levels to assess application security posture.

OWASP Web Security Testing Guide (WSTG)

Official: owasp.org/www-project-web-security-testing-guide/

A comprehensive manual for testing web applications, covering authentication, session management, data validation, and more.

OWASP Mobile Security Testing Guide (MSTG)

Official: owasp.org/www-project-mobile-security-testing-guide/

Provides detailed test cases and methodologies for mobile application security assessments.

General Standards & Frameworks

Penetration Testing Execution Standard (PTES)

Official: pentest-standard.org

Outlines phases of a pentest: pre-engagement, intelligence gathering, threat modeling, vulnerability analysis, exploitation, post-exploitation, and reporting.

Common Weakness Enumeration (CWE)

Official: cwe.mitre.org

A community-driven list of software and hardware weaknesses, aiding in identifying, understanding, and preventing vulnerabilities.

SANS/CWE Top 25 Most Dangerous Software Errors

Official: cwe.mitre.org/top25/

Catalogs the most impactful software errors leading to serious vulnerabilities in applications.

Building Security In Maturity Model (BSIMM)

Official: bsimm.com

Provides a data-driven model of observed software security practices to measure and guide security initiatives.

Open Software Assurance Maturity Model (OpenSAMM)

Official: owasp.org/www-project-OpenSAMM/

Framework to help organizations formulate and implement a strategy for software security tailored to their risk profile.

NIST SP 800-115

Official: csrc.nist.gov/publications/detail/sp/800-115/final

Technical guide for information security testing and assessment, covering network and application-layer evaluations.

WASC Threat Classification

Official: projects.webappsec.org/WASC-Threat-Classification

Categorizes web application threats to help organizations understand and prioritize security risks effectively.
