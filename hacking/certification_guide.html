Cyber Security Certification Guide

Beginner Certifications

CompTIA Security+

Description: Vendor-neutral entry-level certification covering core security functions.

Complexity: Beginner

Skills Required: Basic networking, threat concepts, cryptography fundamentals.

Prerequisites: CompTIA Network+ recommended but not mandatory.

Official: CompTIA Security+

eLearnSecurity Junior Penetration Tester (eJPT)

Description: Hands-on web and network pentesting fundamentals.

Complexity: Beginner

Skills Required: Linux basics, networking, web app basics.

Official: eJPT

Intermediate Certifications

Certified Ethical Hacker (CEH)

Description: Offensive security certification teaching common hacking techniques.

Complexity: Intermediate

Skills Required: Networking, scripting, vulnerability analysis.

Prerequisites: 2 years of security experience recommended.

Official: CEH

CompTIA CySA+

Description: Threat detection, behavioral analytics, and incident response.

Complexity: Intermediate

Skills Required: Security monitoring, SIEM tools, threat intelligence.

Official: CompTIA CySA+

Advanced Certifications

Offensive Security Certified Professional (OSCP)

Description: Practical pentesting certification with 24-hour lab exam.

Complexity: Advanced

Skills Required: Bash, Python, Metasploit, manual exploitation.

Official: OSCP

CISSP

Description: Managerial-level certification covering eight security domains.

Complexity: Advanced

Prerequisites: 5 years of security work experience.

Official: CISSP

Vendor-Specific & Other Certifications

eCPPT

Description: Comprehensive penetration testing process: scoping, enumeration, exploitation, reporting.

Official: INE eCPPT

eCRTP

Description: Red teaming basics: adversary simulation, lateral movement, post-exploitation.

Official: INE eCRTP

eCRTE

Description: Advanced red team operations: opsec, persistence, evasion techniques.

Official: INE eCRTE

OSWP

Description: Wi-Fi network penetration testing with Kali Linux.

Official: OSWP

OSEP

Description: Advanced web attacks and exploitation techniques.

Official: OSEP

OSWE

Description: Deep dive into web application exploit development.

Official: OSWE

OSED

Description: Exploit development methodologies and advanced techniques.

Official: OSED

SecOps Certified SOC Analyst (CSCA)

Description: SOC operations, monitoring tools, incident detection.

Official: SecOps CSCA

SecOps Certified Threat Intelligence Analyst (CTIA)

Description: Threat intelligence lifecycle: collection, analysis, dissemination.

Official: SecOps CTIA

SecOps Incident Response Specialist (CIRS)

Description: Incident response process: triage, containment, eradication, recovery.

Official: SecOps CIRS

GSEC

Description: Security essentials: best practices, policies, and techniques.

Official: GSEC

GPEN

Description: Penetration testing methodologies and tools usage.

Official: GPEN

GCIH

Description: Incident handling and response techniques.

Official: GCIH

GCIA

Description: Network intrusion analysis and threat detection.

Official: GCIA

GWAPT

Description: Web application penetration testing techniques.

Official: GWAPT

Suggested Certification Paths

  • Generalist Track: Security+ → CySA+ → GSEC → CISSP
  • Offensive Specialist: eJPT → eCPPT → OSCP → OSWE → OSED
  • Red Team Track: eCRTP → eCRTE → CIRS → Advanced OPSEC courses
  • SOC/Defensive Track: CSCA → GCIH → GCIA → CISSP
  • Cloud Security Focus: Security+ → AWS Security Specialty → CCSP
